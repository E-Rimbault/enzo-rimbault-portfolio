Langages Informatiques Pratiqués

Durant mon BTS - SIO, j'ai travaillé sur divers langages informatiques essentiels. Découvrez ci-dessous le détail des langages utilisés, ainsi que leur contexte d'application dans mes projets.

HTML

Langage de balisage essentiel pour la structure des pages web.
Utilisé pour créer des sites statiques et dynamiques, assurant la sémantique du contenu.

CSS

Langage de style permettant de définir l’apparence des pages web.
J'ai travaillé sur la mise en page, l'animation et la responsivité pour des interfaces modernes.

JavaScript

Langage de programmation indispensable pour l'interactivité des sites web.
Employé pour la manipulation du DOM, la création d'animations et l'intégration de fonctionnalités avancées, ou encore le fetch de données depuis des fichiers json.

Python

Langage polyvalent, utilisé pour l'automatisation, le traitement des données et le scripting.
Mes projets incluent l'analyse de données et l'automatisation de tâches répétitives.

PHP

Langage de script côté serveur pour le développement web.
Utilisé dans la création de contenus dynamiques et la gestion de bases de données.

SQL

Langage de requête pour manipuler et interroger des bases de données relationnelles.
Employé pour concevoir et gérer des bases de données dans mes projets web.

Java

Langage de programmation orienté objet pour des applications multiplateformes.
Utilisé dans le développement d'applications robustes et évolutives lors de ma formation.
