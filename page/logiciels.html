Logiciels / système d'exploitation Utilisés

Pendant ma formation en BTS SIO, j'ai utilisé plusieurs logiciels essentiels pour le développement, la gestion des bases de données, ainsi que des outils de collaboration et de design.

Logiciels

Eclipse

Un IDE utilisé pour le développement d'applications Java et d'autres langages. Il m'a été utile pour des projets académiques.

EduPython

Un environnement de programmation utilisé pour l'apprentissage de la programmation en Python. Il facilite les débuts avec ce langage.

VSCode

Un éditeur de code léger et puissant, utilisé pour écrire du code dans différents langages tels que Python, JavaScript, et PHP.

GitHub

Plateforme de gestion de versions et de collaboration. J'ai utilisé GitHub pour gérer mes projets et travailler en équipe.

Canva

Outil de design graphique en ligne utilisé pour créer des visuels attrayants, notamment pour des présentations et des sites web.

WAMP

Ensemble de logiciels permettant de créer un serveur web local pour tester des projets PHP et MySQL en local.

phpMyAdmin

Interface web permettant la gestion des bases de données MySQL de manière visuelle et simplifiée.

WordPress

CMS (Content Management System) utilisé pour la création de sites web dynamiques, notamment avec le thème AVADA.

Laravel

Framework PHP moderne utilisé pour développer des applications web robustes et sécurisées en suivant le modèle MVC. Il m’a permis de structurer efficacement mes projets PHP.

Systèmes d’exploitation

Linux

Système d'exploitation open-source, utilisé pour la gestion de serveurs et l'administration système.

Windows

Système d'exploitation utilisé pour la majorité de mes travaux de développement et de gestion informatique.
