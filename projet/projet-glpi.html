Installation complète de GLPI 9.1 sur Ubuntu Server 16.04

Introduction

Ce guide détaillé vous accompagne à travers l’installation complète de GLPI 9.1 sur un serveur Ubuntu 16.04. GLPI est une solution open-source permettant la gestion de parc informatique, la gestion des tickets, des équipements et des utilisateurs.

L’objectif de ce tutoriel est d’expliquer chaque étape avec clarté afin que même un débutant puisse déployer un environnement professionnel et fonctionnel.

Vous installerez successivement : Apache, MySQL, PHP, phpMyAdmin, GLPI ainsi que le plugin FusionInventory utilisé pour automatiser l'inventaire matériel et logiciel du réseau.

Étapes d'installation et configuration

1. Connexion et Configuration du Compte ROOT

Avant toute manipulation système, vous avez besoin des droits administrateur.

  • Connectez-vous avec l’utilisateur créé pendant l’installation d’Ubuntu.
  • Passez en mode superutilisateur : sudo su
  • Définissez un mot de passe pour root : sudo passwd root
  • Redémarrez le serveur pour appliquer les changements : reboot

2. Configuration d'une Adresse IP Statique

Pour éviter que l’adresse du serveur ne change, il est essentiel de définir une IP fixe.

sudo vim /etc/network/interfaces
# Exemple :
auto ens32
iface ens32 inet static
 address 192.168.0.243
 netmask 255.255.255.0
 gateway 192.168.0.254
 dns-nameservers 192.168.0.1

Redémarrez ensuite : reboot

3. Installation et Configuration de SSH

SSH permet l’accès distant sécurisé au serveur, pratique pour administrer la machine sans écran.

  • Installer OpenSSH : sudo apt-get install openssh-server
  • Dans /etc/ssh/sshd_config, activez la connexion root :
PermitRootLogin yes

Puis redémarrez le service SSH.

4. Installation d'Apache (Serveur Web)

Apache affichera l’interface web de GLPI.

sudo apt-get update
sudo apt-get install apache2
sudo vim /etc/apache2/apache2.conf
# Ajoutez :
ServerName 192.168.0.243
sudo systemctl restart apache2

5. Configuration du Pare-feu (UFW)

On autorise uniquement les services nécessaires : HTTP, HTTPS et SSH.

sudo ufw enable
sudo ufw allow in "Apache Full"
sudo ufw allow 22/tcp
sudo ufw status verbose

6. Installation de MySQL

MySQL stockera toutes les données de GLPI (tickets, utilisateurs, inventaire…).

sudo apt-get install mysql-server

7. Installation de PHP

PHP est nécessaire pour faire fonctionner GLPI côté serveur.

sudo apt-get install php libapache2-mod-php php-mcrypt php-mysql php-curl

8. Installation de phpMyAdmin

phpMyAdmin permet de gérer la base MySQL via une interface graphique.

sudo apt-get install phpmyadmin apache2-utils
  • Sélectionnez Apache2.
  • Répondez « NON » à la configuration automatique.
  • Accédez via : http://IP_DU_SERVEUR/phpmyadmin

9. Installation de GLPI 9.1

GLPI est maintenant prêt à être installé sur la machine.

cd /tmp
wget https://github.com/glpi-project/glpi/releases/download/9.1/glpi-9.1.tar.gz
cd /opt
sudo tar -xvzf /tmp/glpi-9.1.tar.gz

Configuration Apache

sudo vim /etc/apache2/conf-available/glpi.conf
Alias /glpi /opt/glpi

  DirectoryIndex index.php
  Options FollowSymLinks
  AllowOverride Limit Options FileInfo
  Require all granted

Activation

sudo ln -s ../conf-available/glpi.conf /etc/apache2/conf-enabled/glpi.conf
sudo service apache2 restart

Permissions

chmod -R 777 /opt/glpi/files
chmod -R 777 /opt/glpi/config

Installation via navigateur

Rendez-vous sur http://IP_DU_SERVEUR/glpi puis suivez les étapes classiques.
