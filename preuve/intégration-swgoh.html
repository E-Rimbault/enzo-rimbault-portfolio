Tests d’intégration et d’acceptation d’un service

Projet : SWGOH API Viewer

Description du projet

Ce projet consiste à exploiter l'API publique de swgoh.gg afin d'afficher dynamiquement des données sur les personnages du jeu Star Wars Galaxy of Heroes.

URL du projet en ligne : Voir le site

Détails complets du projet : Page descriptive du projet

Tests d’intégration

J’ai intégré les données récupérées depuis l’API swgoh.gg dans mon site. Les tests d’intégration ont permis de vérifier :

  • La bonne réception des données JSON via les requêtes HTTP (AJAX ou Fetch).
  • Le bon affichage dynamique des données dans le DOM (noms, images, stats, etc.).
  • La compatibilité entre les données API et les composants visuels du site.

Tests d’acceptation

J’ai également effectué des tests d’acceptation pour valider la conformité fonctionnelle avec les besoins utilisateurs :

  • Affichage fluide des informations recherchées par l’utilisateur.
  • Navigation simple et intuitive entre les fiches personnages.
  • Compatibilité mobile et responsive.
