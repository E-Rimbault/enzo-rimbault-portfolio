Preuve de Compétence : Veille Technologique

Dans le cadre de mon BTS SIO, j’ai mis en œuvre une veille informationnelle sur les thèmes liés à la cybersécurité, afin de rester informé des évolutions du secteur. Cette veille constitue une compétence professionnelle essentielle, démontrant ma capacité à m’informer de manière autonome et à sélectionner les informations pertinentes.

Choix des Thèmes

J’ai choisi de structurer ma veille autour de quatre grandes thématiques :

  • Cyberattaques : pour comprendre les types d’attaques actuelles et leur impact sur les organisations.
  • Fuites de données : afin d’analyser les conséquences des pertes d’informations sensibles.
  • Actualités de la CNIL : pour rester à jour sur la régulation et la protection des données personnelles.
  • Évolutions en cybersécurité : pour suivre les nouvelles technologies, pratiques et outils de sécurité.

Outils Utilisés

Pour réaliser cette veille, j’ai utilisé les outils suivants :

  • Flux RSS : permettant d’agréger les contenus de plusieurs sites spécialisés (ZDNet, Numerama, The Hacker News, etc.).
  • API rss2json : pour convertir les flux RSS en format JSON, exploitable en JavaScript.
  • Script JavaScript personnalisé : qui trie les articles par thème et date, les affiche dynamiquement dans la page veille-technologique.html.

Preuve

La page Veille Technologique montre concrètement cette veille automatisée et dynamique. Elle prouve ma capacité à :

  • identifier les sources fiables,
  • structurer une veille thématique,
  • automatiser la collecte et l'affichage d'informations,
  • mettre en œuvre des outils web et des scripts API.
