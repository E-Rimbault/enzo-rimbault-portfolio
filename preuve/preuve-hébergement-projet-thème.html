Déploiement du site SWGOH Wiki sur GitHub Pages

Présentation du projet

Le projet SWGOH Wiki est un site web statique qui présente des informations détaillées sur les personnages du jeu mobile Star Wars: Galaxy of Heroes (SWGOH). Il permet aux joueurs de consulter les kits des personnages pour mieux comprendre leurs compétences et pouvoirs.

Ce projet m’a permis de renforcer mes compétences en structuration de contenu, en organisation d’un site documentaire et en mise en place d’une architecture claire permettant une navigation fluide. L’objectif était d’offrir une interface simple et intuitive, adaptée à un public cherchant avant tout une information rapide et fiable.

Le projet repose sur :

  • HTML pour la structure.
  • CSS pour la mise en forme.
  • JavaScript pour l'interactivité.

Outils utilisés

  • Jeux mobiles — Informations provenant du jeu SWGOH.
  • Site officiel — Star Wars: Galaxy of Heroes.
  • Visual Studio Code — Éditeur de code utilisé pour le développement.
  • Google Search Console — Référencement et indexation du site.

Le déploiement sur GitHub Pages

J'ai utilisé GitHub Pages pour héberger gratuitement ce site statique. Cette solution est idéale pour les projets personnels, portfolios et documentations, car elle permet de mettre en ligne un site immédiatement sans devoir configurer un serveur.

Les étapes :

  1. Création d'un dépôt GitHub et importation du projet.
  2. Activation de GitHub Pages via la branche main ou gh-pages.
  3. Configuration du chemin d’accès afin que les fichiers HTML soient correctement interprétés.
  4. Vérification que le site s’affiche correctement et optimisation du temps de chargement.
  5. Le site est accessible ici : https://e-rimbault.github.io/swgoh-kit/.

Ce déploiement m’a permis de mieux comprendre le fonctionnement des pages statiques hébergées par GitHub, ainsi que la gestion du versionnement dans un contexte de projet web.

Référencement sur Google

Grâce à Google Search Console, j'ai amélioré l'indexation et la visibilité du site. Cela m’a notamment permis d’analyser la manière dont Google lit et comprend mes pages, d’identifier d’éventuelles erreurs d’exploration et d’optimiser le référencement global.

J’ai également travaillé sur les métadonnées (description, titre, meta-tags structurels) afin d’améliorer l’apparence du site dans les résultats de recherche et ainsi maximiser les chances d’être visité.
