Participation à l’évolution d’un site Web exploitant des données API

Description du projet

Le projet vise à construire une interface dynamique en consommant des données de l’API publique de swgoh.gg. Cette évolution permet une mise à jour automatique des informations sans modifier le code manuellement.

🔗 Site en ligne : SWGOH API Info

🔗 Page projet : Voir la fiche détaillée

Exploitation de l’API publique

Grâce à l’API, le site affiche des données toujours à jour : nouveaux personnages, mises à jour de statistiques, etc. Cela permet au site de rester utile pour les utilisateurs sans intervention manuelle.

  • 🔁 Intégration continue de données actualisées
  • 📈 Affichage de données précises issues du jeu
  • 🧩 Requête dynamique via JavaScript et gestion JSON

Améliorations futures

Des fonctionnalités peuvent encore être ajoutées :

  • 🔎 Filtres et recherches plus avancées
  • 💡 Ajout de favoris ou de comparateur de personnages
  • 🌐 Multilingue (FR/EN)
